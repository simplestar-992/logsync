# LOGSYNC

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/logsync.git
cd logsync
go build -o logsync .
```

## Usage

```bash
./logsync --help
```
